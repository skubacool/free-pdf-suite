Several pictures, one image file

Sometimes you need several pictures to become one: a chat conversation captured across three screenshots, the front and back of an ID, a before-and-after pair, or a set of receipts you want to send as a single file. This tool draws them onto one canvas and gives you back a single image.

Choose stacked to place them one above the other, side by side for a horizontal strip, or grid to arrange them in rows and columns. Add spacing and a background colour, and export as JPG or a transparent PNG.

How to use it

  1. Drop two or more images - they merge in the order you add them.
  2. Pick the layout (stacked, side by side, or grid) plus spacing, background and format.
  3. Click Merge Images, check the preview, and download.
🔒 100% private. Your images are combined entirely in your browser - they are never uploaded to any server.

Need each picture on its own page instead? Use JPG to PDF. Want to shrink the result? Try Compress Image.

Frequently asked questions

How is this different from JPG to PDF?

JPG to PDF puts each image on its own PDF page. This tool combines the images into a single image file (JPG or PNG) that you can post, message or print as one picture.

Can I control the order?

Yes - images merge in the order you add them. Drop them in the order you want, or use Clear to start over and re-add them.

What if my images are different sizes?

Vertical mode scales every image to the widest one, horizontal mode scales to the tallest, and grid mode fits each image inside a uniform cell so nothing is distorted.

Can I keep a transparent background?

Yes - choose PNG output and tick Transparent background. JPG cannot store transparency, so it always uses the background colour.

Are my images uploaded?

No. The merge is drawn on a canvas in your browser, so your images never leave your device and it works offline.
