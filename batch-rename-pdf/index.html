How to Batch Rename PDF Files

Organizing client files, invoices, or receipts requires consistent naming conventions. With this tool, you can upload multiple PDFs at once and apply a structured naming template. You can use variables like {n} to insert a sequential number, or {orig} to keep the original file name. The renamed files are zipped together for easy download.

Batch Rename PDFs — Frequently Asked Questions

What variables can I use in the naming pattern?

You can use {n} for a sequential number (starting at 1) and {orig} for the original name of the file (excluding extension). For example, "Receipt_{n}" will name them Receipt_1.pdf, Receipt_2.pdf, etc.

How many files can I rename at once?

Since renaming runs locally on your browser, there is no file count limit. It can rename hundreds of files in seconds.

Is my document uploaded to a server?

No. Everything runs locally in your browser. Your files never leave your device.
