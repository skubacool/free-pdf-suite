Set the exact dimensions you need

Whether you need a 1080-pixel-wide photo for a website, a smaller avatar, or a half-size copy to email, this tool scales your images to precise dimensions. By default it keeps the original aspect ratio so nothing looks stretched.

Choose to scale by a percentage, lock a target width or height, or set an exact width × height. Process one image or a whole batch at once.

How to use it

  1. Drop your images (JPG, PNG or WebP).
  2. Pick how to resize - percentage, a fixed width or height, or exact dimensions.
  3. Enter the value and click Resize Images, then download.
🔒 100%% private. Your images are processed entirely in your browser using the Canvas API - they are never uploaded to any server. You can even run this tool offline.

Frequently asked questions

Will resizing keep the aspect ratio?

Yes - the Percentage, Width and Height modes all preserve the original proportions. Only the exact width by height mode can stretch an image if your numbers do not match its ratio.

Can I enlarge a small image?

You can, by setting a larger size or a percentage above 100%, but enlarging cannot add detail that is not there, so big upscales look soft.

What formats can I resize?

JPG, PNG and WebP. The output keeps the same format as the original, so PNG stays PNG and transparency is preserved.
